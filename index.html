tricks

  • add alpha channel to a background layer.This will allow that the background can be removed
  • Zoom tool can select a specific area then zoom on it so that fill the screen
  • RGB for screen
  • CMY for printer
  • If you hold CTRL your circle center in clicked point
  • For select all parts without a part :
    1. select the part
    2. select >> sharpen
    3. select >> float
    4. make new layer in float
    5. select all area without our part by magic selection
  • Pencil tool and make straight line
    • make a point somewhere HOLD-SHIFT your line is ready
    • make a point somewhere HOLD-SHIFT-CTRL your line is ready for each devide integer
  • In Selection tool:
    • to change the size of a selection just select a corner of that
    • by press BACKSPACE you can undo last doing on selection
    • active the fixed option,set aspect ratio for that,then write 1:1
    • For moving a selection:
      1. CTRL x
      2. CTRL v
      3. make a new layer
      4. Move
  • In Path tool :
    • cursor on a point can change its position
    • cursor on a path can change the path style
    • cursor on edit mode can change the path by change the degree of one of the hands of a point
    • you can remove a path by path panel next to the layers panel
    • if you want to make changes on a previous path just select it from path panel
    • you can make a selection from a path and invert it
    • after shape your polygan ,press ENTER to complete the action
